# 📄 requirements.md

## [개요]

**JD-Vector: AI 기반 직무 적합도 분석 및 커리어 로드맵 서비스**

- **목표**: 사용자의 기술 스택(Vector A)과 채용 공고의 요구사항(Vector B) 사이의 거리를 계산하고, 그 간극(Delta)을 메우기 위한 최적의 학습 로드맵을 제시함.
- **배포**: Frontend (Vercel), Backend (FastAPI - Fly.io 또는 Supabase Edge Functions), Database (Supabase)
- **사용자**: 자신의 현재 위치를 객관적으로 파악하고 전략적으로 취업을 준비하고 싶은 주니어 개발자.

---

## [요구 사항]

### 1. 핵심 기능 (Core Features)

- **멀티 소스 데이터 업로드**:
- 사용자의 이력서(PDF) 및 프로젝트 경험(Text/Markdown) 입력.
- 대상 채용 공고(JD)의 텍스트 붙여넣기 또는 PDF 업로드.

- **RAG(Retrieval-Augmented Generation) 기반 분석**:
- LangChain을 활용하여 JD에서 핵심 기술 스택, 우대 사항, 도메인 지식 추출.
- 사용자의 경험 데이터와 JD 요구사항 간의 **벡터 유사도(Cosine Similarity)** 계산.

- **AI 에이전트 가이드**:
- 부족한 역량을 보완할 수 있는 구체적인 프로젝트 기능 추가 제안.
- "이 기업은 Redis 경험을 중시하니, 기존 프로젝트에 캐싱 로직을 추가해보세요"와 같은 맞춤형 조언.

### 2. UI/UX 및 시각화

- **대시보드 형태의 결과창**:
- **Radar Chart(역량 오각형)**: 언어, 프레임워크, CS 기초, 협업 툴, 도메인 지식별 매칭도 시각화.
- **Match Score**: 전체적인 적합도를 퍼센트(%)와 등급으로 표시.

- **인터랙티브 로드맵**:
- AI가 제안하는 할 일 목록(To-Do)을 체크리스트 형태로 제공.

- **반응형 디자인**: Tailwind CSS를 활용한 모던하고 깔끔한 개발자용 UI (다크 모드 기본 지원).

### 3. 기술 스택 및 데이터 처리

- **Frontend**: React, TypeScript, Tailwind CSS, Recharts (시각화), Lucide React (아이콘).
- **Backend**: FastAPI (Python), LangChain (AI 오케스트레이션).
- **AI/LLM**: OpenAI API (GPT-4o mini) 또는 Ollama (로컬 테스트용).
- **Database/Auth**: Supabase (PostgreSQL + pgvector를 활용한 벡터 검색 연동 가능성 염두).
- **Data Analysis**: CDS 실습 경험을 살린 유사도 분석 알고리즘 적용 및 지표 산출.

### 4. 개발 컨벤션 및 관리

- **Git**: Conventional Commits 준수 (`feat:`, `fix:`, `docs:`, `ai:`, `data:` 등).
- **Directory**: Monorepo 구조 지향 (Frontend와 Backend 분리).
- **Code Quality**: TypeScript를 통한 타입 안정성 확보 및 AI 리터치를 고려한 모듈화된 코드 작성.
